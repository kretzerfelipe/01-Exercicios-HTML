React

React é uma biblioteca JavaScript de código aberto para criar interfaces de usuário, mantida pelo Facebook. É usada principalmente para construir single-page applications (SPAs) e interfaces móveis.

Seu principal recurso é a capacidade de criar componentes reutilizáveis, tornando o desenvolvimento mais eficiente e organizado.

Criado em 2011 pelo Facebook, com a criação de views declarativas e baseando-se em componentes, possuía o intuito de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social e melhorar a manutenção de código.
